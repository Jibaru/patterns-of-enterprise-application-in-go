# Template View

Renders information into HTML by embedding markers in an HTML page.

## Example description

By using Go's templating engine as a template view pattern, we demostrate how to render a book details, also using a book helper.

## How to run?

1. Run (make sure you have 8080 port available, if not, change it in the main function):

```
go run ./cmd/app/main.go
```
